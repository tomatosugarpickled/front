# C&M 스타트업 플랫폼 프로젝트 개요

## 1. 프로젝트 비전

**"인재와 자본의 간극을 잇다"**

본 프로젝트는 초기 창업 생태계의 핵심적인 불일치 문제를 해결합니다:

1.  **인재 미스매치:** 개발 능력이 없는 마케터와 비즈니스 감각이 부족한 개발자의 만남.
2.  **투자의 불투명성:** 초기 팀을 평가할 때 객관적인 지표(팀 경력 합계, 기술 스택 등)의 부재.

우리 플랫폼은 개인이 공동 창업자를 찾고, 팀이 스타트업으로 발전하며, 자본을 유치하는 통합 생태계입니다.

## 2. 핵심 가치 제안

| 대상            | 페인 포인트 (Pain Point)              | 솔루션 (Solution)                                                                  |
| :-------------- | :------------------------------------ | :--------------------------------------------------------------------------------- |
| **개인**        | "마케팅은 자신 있는데 개발을 못해요." | **파트너 매칭:** 부족한 역량을 채워줄 공동 창업자 탐색.                            |
| **팀**          | "팀은 꾸렸는데 아직 회사는 아니에요." | **스타트업 등록:** 팀의 정체성을 공식화하고 이력을 쌓는 단계.                      |
| **투자자/기업** | "이 팀이 정말 실력이 있을까?"         | **데이터 기반 기업 페이지:** 포트폴리오, 기술 스택, 팀원 총 경력 등을 한눈에 파악. |

---

## 3. 주요 기능 및 모듈

### A. 팀 빌딩 & 파트너 매칭 (현재 단계)

- **목적:** 실행을 위한 최소 단위(팀) 결성.
- **파트너 찾기:**
    - 직무별 상세 필터링 (예: "풀스택 개발자", "퍼포먼스 마케터").
    - 경력 및 선호 근무 방식(원격, 하이브리드) 기반 매칭.
- **팀 리스트:**
    - 팀원을 모집 중인 팀들의 목록.
    - 현재 팀 구성 현황 및 영입 필요한 포지션 노출.

### B. 스타트업 생태계 (다음 단계)

- **스타트업 등록:**
    - '팀' 단계에서 정식 '스타트업'으로 전환하는 과정.
    - 사업성 입증을 위한 검증 단계.
- **기업 페이지 (데이터 중심 프로필):**
    - **종합 포트폴리오:** MVP, 시제품, 라이브 서비스 갤러리.
    - **팀 지표:**
        - **총 경력:** "팀원 도합 경력 15년"
        - **역량 비율:** "개발 3 : 기획 2 : 디자인 1"
    - **기술 스택:** 사용 중인 기술 시각화 (React, Java, AWS 등).
    - **비즈니스 프로필:** 비즈니스 모델 캔버스, 타겟 시장, 성장 전략 명시.

### C. 펀딩 & 투자 시스템 (미래 단계)

- **목적:** 준비된 스타트업과 자본의 연결.
- **펀딩 모델:**
    - **대상:** CVC(기업 주도형 벤처 캐피털), 엔젤 투자자, 엔터프라이즈 파트너.
    - **메커니즘:** 투자자는 "기술 스택"이나 "사업 분야" 필터로 기업을 검색하고 펀딩 제의.
    - **펀딩 상태:** 부트스트래핑, 시드(Seed), 프리-A(Pre-A) 등 자금 조달 의향 표시.

## 4. 유저 시나리오

1.  **아이디어:** **마케터**가 SaaS 아이디어를 가지고 있지만 개발자가 없음.
2.  **매칭:** 플랫폼에서 사이드 프로젝트에 관심 있는 **시니어 개발자**를 **파트너 찾기**로 탐색.
3.  **결성:** 둘이 뭉쳐 **팀**을 만들고, **팀 리스트**를 통해 주니어 디자이너 영입.
4.  **성장:** 3개월간 MVP 개발 후, **스타트업 등록** 진행.
5.  **쇼케이스:** **기업 페이지** 생성 및 데이터 입력:
    - _스택:_ Flutter, Firebase.
    - _팀 경력:_ 도합 12년.
    - _포트폴리오:_ 데모 비디오 업로드.
6.  **펀딩:** "AI 기반 SaaS"를 찾던 **대기업 파트너**가 해당 페이지를 방문, 기술 스택과 팀 역량을 확인 후 **시드 펀딩** 제안.
